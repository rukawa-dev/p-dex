# 지역 도감 추가하는 방법
https://docs.google.com/spreadsheets/d/1k1QfNi_DtKKBe6IG07Ol68P-SiKTGzwml36daXYn8jQ/edit?gid=0#gid=0  
이 프로젝트는 구글 시트의 데이터를 기반으로 지역 도감을 동적으로 생성합니다. 새로운 지역 도감을 추가하려면 아래 단계를 따르세요.

## 1. 지역도감 시트 정보 설정

`지역도감_시트정보.json` 파일에 새로운 도감 정보를 추가합니다.

```json
{
  "key": "unique-id",          // 도감의 고유 키 (예: paldea, za) -> 파일명은 자동으로 pokemon-unique-id.json이 됩니다.
  "game": "게임 이름",          // 사이드바 카테고리 그룹 이름
  "map_url": "지도 URL",        // (선택) 외부 지도 링크 버튼 URL
  "name": "도감 이름",          // 도감 버튼에 표시될 이름
  "gid": "구글시트_GID"         // 구글 시트 탭의 고유 ID (URL의 gid 부분)
}
```

## 2. 구글 시트 준비

등록할 구글 시트는 다음과 같은 형식을 갖춰야 합니다.
- **포켓몬 이름**이 포함된 열(Column)이 반드시 존재해야 합니다.
- 스크립트가 자동으로 '포켓몬 이름', '이름', 'Name' 등의 헤더를 찾아서 포켓몬 번호와 매칭합니다.

## 3. 데이터 갱신 명령어 실행

설정을 마친 후 터미널에서 아래 명령어를 실행하여 데이터를 생성합니다.

```bash
# 새로운 데이터만 가져오기
npm run data:all

# 구글 시트 데이터를 강제로 다시 긁어오기
npm run data:all:refresh
```

## 4. 확인

- `npm run dev`를 실행 중이라면 페이지를 새로고침하면 사이드바에 자동으로 새로운 카테고리와 도감 버튼이 나타납니다.
- `map_url`을 넣었다면 도감 이름 옆에 지도 아이콘 버튼이 생성됩니다.
- Lucide 아이콘 라이브러리가 적용되어 있어 디자인이 자동으로 일관성 있게 유지됩니다.
